## Installation:
1. `Clone this project`
2. `cd pomodoro`
3. `npm install`
4. `expo install expo-font @expo-google-fonts/montserrat`

## Running The Project
`npm start`

## Features:
- Pomodoro Timer with vibration and sound
- Visual feedback when the timer is running
